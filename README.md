## 🚀 Getting Started

To run the examples:

1. Clone the repository:

   ```bash
   git clone https://github.com/Rafsan12/typeScript.git
   ```

2. Navigate to a module folder:

   ```bash
   cd typeScript/module-1
   ```

3. Install dependencies (if any):

   ```bash
   npm install
   ```

4. Compile and run TypeScript files:
   ```bash
   tsc filename.ts
   node filename.js
   ```
